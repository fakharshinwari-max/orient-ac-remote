Captured remote codes

Every button below was captured from the real Orient AC remote using capture/OrientIRCapture.ino. Tap "Copy" and paste the pattern into IR Blaster Remote → your remote → Add Button → Raw Signal.

No codes captured yet. Run the capture sketch, save a capture_log.txt, then run python scripts/parse_capture.py capture_log.txt — this page updates automatically from orient_ac_codes.json.

Frequency for all captured signals defaults to 38000 Hz unless noted otherwise per button. If a button doesn't respond on the AC after pasting, re-capture it slightly closer to the receiver — weak signal strength is the most common cause of a bad capture, not a wrong protocol.
